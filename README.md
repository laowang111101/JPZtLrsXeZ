# 超市进销存管理系统 java969

## 项目概述

本项目是一款基于springboot与vue的超市进销存管理系统。通过此系统，可以实现高效的超市库存管理与流程控制，提高工作效率。

## 技术栈

- 后端：Java语言，springboot框架
- 前端：vue
- 数据库：MySQL

## 功能模块

- 用户登录
- 个人中心
- 管理员管理
- 用户信息管理
- 仓库信息管理
- 物品分类管理
- 物品信息管理
- 商品出入库管理

## 系统角色

- 普通用户：进行登录，查看与修改个人中心信息
- 管理员：管理用户信息，仓库信息，物品分类与物品信息，处理商品出入库操作

## 运行环境

- 后端：Java环境（推荐JDK 1.8+）
- 前端：基于Web浏览器（推荐使用Chrome、Firefox等现代浏览器）
- 数据库：MySQL（推荐5.7及以上版本）

## 部署步骤

1. 搭建Java开发环境，并安装MySQL数据库
2. 克隆或下载项目源代码
3. 导入数据库文件，并配置数据库连接信息
4. 编译并运行后端程序
5. 前端通过npm或yarn安装依赖，然后运行构建命令，启动前端服务
